# 🚀 Mini File Manager (CLI-Based)

## 📌 Overview

Mini File Manager is a command-line application built using Python that allows users to manage files and directories efficiently. It provides essential file system operations through a simple, menu-driven interface.

This project demonstrates practical usage of Python's built-in modules for interacting with the operating system.

---

## ✨ Features

* 📂 List all files and folders in a given directory
* 📁 Create new folders
* ❌ Delete files or empty folders
* ℹ️ View file details (creation & modification time)
* 🔁 Interactive menu-based system

---

## 🛠️ Tech Stack

* Python 3
* `os` module (file & directory operations)
* `datetime` module (file metadata handling)
